## 2. UI/UX · 비주얼 (Visual Spec)

### 2.1 감성 · 컬러

| 토큰 | 값 | 용도 |
|------|-----|------|
| Paper | `#FDFBF7` | 일력 패드 / 한지 질감 |
| Ink | `#2C2C2C` | 본문·먹색 텍스트 |
| Crimson | Deep Red | 상단 마운트, 요일 바, 양력 큰 숫자, 활성 탭 |
| Gold | 금박 톤 | 인감·링·길상어 |

- [x] 종이 텍스처(스펙클) 에셋 확정
- [x] Sans(라벨) + 전통/고대비 숫자 폰트 페어 확정
- [x] 상단 70% 코어 / 하단 30% 위젯 트레이 비율 고정

### 2.2 레이아웃 (메인)

- [x] 상단 빨간 마운트: `PHÚ QUÝ` / `VINH HOA` / `CÁT TƯỜNG` + 금링 3개
- [x] 좌: 양력연·간지연·월 / 우: 금색 원형 인감
- [x] 중앙: 음력일 · 요일 빨간 바 · 음력월 · **양력 큰 숫자** · 세로 한자
- [x] 하단 좌: 년/월/일/시 간지 · 중: 명언 · 우: 황도시간·순리
- [x] 페이지 푸터: 인장 3 + `TỨ QUÝ BÌNH AN`
- [x] 위젯 트레이 카드 + 하단 탭 4 (`HÔM NAY` / `TỬ VI` / `TIỆN ÍCH` / `CÁ NHÂN`)

### 2.3 인터랙션

- [x] Tear-off: 좌스와이프 → 찢김 애니 + 바스락 사운드 + Haptic
- [ ] 스탬프: 탭 → “쿵” + 인장 번짐 + 해당 날짜 DB 저장
- [x] 날짜별 독립 페이지: 과거로 넘기면 그날짜 기록 복원
